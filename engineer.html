Projects

Justin — Elastic Flink Scaler
Systems · Java

Built a hybrid CPU/memory auto-scaling system for Apache Flink jobs. Justin monitors per-operator resource usage and dynamically resizes task slots without pipeline restarts, achieving significant resource savings on real production workloads.

🏆 IFIP DAIS'25 Best Paper 🏆 IFIP DisCoTec'25 Best Paper
Apache FlinkJavaKubernetesResource Mgmt
GitHub 📄 Paper (DAIS '25)
Charon — Multi-Query Scheduler
Systems · Java

Designed and implemented a fine-grain resource allocator and task placement optimizer for multi-query Flink deployments. Charon minimises resource interference between co-located jobs using a bin-packing-style scheduler with live telemetry feedback.

Apache FlinkSchedulingJavaKubernetes
GitHub 📄 Paper (SAC '26)
Alloy — Transparent Storage Proxy
Systems · Envoy

Built a transparent proxy layer (on top of Envoy) that decouples Apache Flink pipelines from their storage backends. Alloy intercepts storage calls without pipeline modifications, enabling hot-swapping of backends and seamless A/B testing of storage systems.

Envoy ProxyApache FlinkApache KafkaGo
GitHub 📄 Paper (DEBS '26)
StreamBed — Capacity Planner
Tools · Python

Built a capacity planning tool for stream processing clusters. StreamBed takes workload characterization data (from Kafka/Flink telemetry) and predicts resource requirements, enabling operators to provision clusters proactively rather than reactively.

PythonApache KafkaApache FlinkModelling
GitHub 📄 Paper (DEBS '24)
